支持多种常见场景，可以通过命令行参数灵活配置。

（二）使用方法

参数说明：

  • --follow-links：启用递归抓取模式
  • --max-pages N：最多抓取 N 篇文章（默认 20）
  • --max-depth N：递归深度（1=只抓直接链接）
  • --no-same-domain：不限制同域名（默认限制）
  • --exclude PATTERN：排除包含特定模式的 URL
  • --format txt/json/md：输出格式
